# Features

The **CandelaCharts SFP Model** is a structured toolkit designed to detect and manage high-probability Swing Failure Patterns (SFPs).

* **SFP Detection**
* **Trendlines and Regression Channel**
* **Lower Timeframe Imbalance (FVG)**
* **Higher Timeframe (HTF) Candle Groups**
* **SFP Lifecycle Tracking**
* **Dashboard**

It combines multi-length pivot detection, dynamic boundary systems, higher-timeframe context, and full lifecycle tracking to provide clear and actionable chart feedback.
