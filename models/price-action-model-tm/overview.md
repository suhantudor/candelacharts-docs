# Overview

The Price Action Model is a powerful and sophisticated indicator built for TradingView. It empowers traders by automating the detection of high-probability trading setups based on core price action principles, liquidity concepts, and market structure analysis.

This documentation provides comprehensive details on how to use the Price Action Model effectively, understand its terminology, and leverage its extensive feature set to improve your trading edge.

Unlike traditional lagging indicators (like moving averages or RSI) that tell you what _has_ happened, the Price Action Model focuses on **what the market is doing right now**. By identifying where liquidity rests and how structure is forming in real-time, the model provides actionable insights that align with how institutional participants trade.
