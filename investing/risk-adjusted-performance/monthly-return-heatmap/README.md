# Monthly Return Heatmap

The **CandelaCharts - Monthly Returns Heatmap** is a powerful visual tool designed to help traders and investors analyze the historical monthly performance of an asset.&#x20;

By aggregating past returns into an intuitive color-coded heatmap, this indicator allows you to instantly recognize recurring seasonal patterns, historically strong or weak months, and overall year-to-date performance.
