# Multi-Asset Monthly Returns Heatmap

The **CandelaCharts - Multi-Asset Monthly Returns Heatmap** is an advanced analytical tool designed for macro investors, portfolio managers, and traders who need to track and compare the historical monthly performance across a broad basket of assets.&#x20;

By aggregating data across multiple markets simultaneously, this indicator provides a bird's-eye view of seasonal trends and relative strength.
