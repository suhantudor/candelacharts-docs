# Confluences

The Macro Liquidity Gap provides the ultimate fundamental backdrop for technical traders:

* **Major Support Levels**: If an index hits a multi-year technical support level _while_ deep in a Liquidity Buy Zone, the confluence provides an asymmetrical risk-reward setup.
* **Trend Reversals**: When combining this indicator with moving averages or oscillators (like RSI), wait for the Macro Liquidity Gap to exit the Buy Zone and print a Buy Signal to confirm that technical momentum is now aligning with underlying fundamental liquidity.
