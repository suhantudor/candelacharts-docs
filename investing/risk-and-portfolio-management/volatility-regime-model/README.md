# Volatility Regime Model

The **CandelaCharts - Volatility Regime Model** is an advanced framework that analyzes market volatility as a series of distinct regimes rather than an absolute oscillator.&#x20;

By combining VIX thresholds, time-decay persistence logic, and cross-asset validation, it helps traders determine when a volatility spike is just "noise" to be faded, or a systemic "shift" to respect.
